# Stock Count — Quick Guide

Reconcile what's physically on the shelf against system stock for **your branch**. Counting doesn't change stock directly — it records the difference as an adjustment when you submit.

## Option A: Count by hand

1. Open the **Count** tab.
2. Find products (search, or filter by category/brand).
3. Type the quantity you physically counted next to each item.
4. The delta (counted − system) shows live. Repeat for every item you counted.
5. Click **Submit**.

## Option B: Count from photos

1. Open the **Photos** tab and upload photos of your count sheet.
2. The app reads each row and matches it to a product. Rows it isn't sure about go to the **review queue**.
3. For each review row, pick one:
   - **Accept** — use the suggested product.
   - **Link** — attach the row to a different product.
   - **Edit** — fix the text/quantity that was misread.
   - **New item** — create a product that doesn't exist yet, this item was not in the system when looked against the items in the list and now you want to add to the system plus the quantity taken.
   - **Skip** — don't count this row.
4. Once the queue is clear, click **Submit**.

> Tip: You can mix both — photos fill in counts automatically, and you can still type or adjust on the Count tab before submitting.

## What Submit does

For each counted item it writes an adjustment so system stock **becomes your counted number**.

- Counted 8, system said 10 → stock is set to **8** (−2 adjustment).
- It **replaces** the total, it does not add on top of existing stock.
- Items where your count matches the system are left unchanged.

After submitting you get a **report** (expected / counted / variance) that you can export as CSV or PDF.

## Good to know

- Stock is **per branch** — you're only counting your own branch's stock.
- Accepting/linking a row only fills in the *counted quantity*. Nothing changes in inventory until you **Submit**.
- Photo drafts and your progress are saved, so you can close the page and resume the same count later.

## Add missing row and New Item buttons
- Add missing row = "the photo missed a whole line — let me type in a missing row."
- New Item = "this row's product doesn't exist in the catalog — create the product."
